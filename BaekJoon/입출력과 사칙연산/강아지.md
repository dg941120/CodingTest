**문제** :아래 예제와 같이 개를 출력하시오.
|\_/|
|q p|   /}
( 0 )"""\
|"^"`    |
||_/=\\__|


```
public class App {
    public static void main(String[] args) {
        System.out.println("|\\_/|");
        System.out.println("|q p|   /}");
        System.out.println("( 0 )\"\"\"\\");
        System.out.println("|\"^\"`    |");
        System.out.println("||_/=\\\\__|");
    }
}

```


메모
```
자바에서 \는 \\로 표기해주어야함
자바에서 "는 \"로 표기해주어야함
```
