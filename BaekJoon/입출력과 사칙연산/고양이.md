**문제** :아래 예제와 같이 고양이를 출력하시오.
\    /\
 )  ( ')
(  /  )
 \(__)|


```
public class App {
    public static void main(String[] args) {
        System.out.println("\\    /\\");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");
    }
}
```


메모
```
자바에서 \는 \\로 표기해주어야함
```
